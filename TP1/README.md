# Travail Pratique 2
Ce dépôt contient le code d’un projet web fait dans le cours de Perspective Professionnelles.
## Faire un _commit_ et un _push_ dans _Visual Studio Code_
Voici comment faire un changement sur ce projet, et l’envoyer sur GitHub. 

1. Sauvgarder les modifications fait sur VSC avec Ctrl+S 
2. Sur Github Desktop, donner un nom a vos modifications sur la "Side Bar"(_et optionellemnt une description_)
3. Commiter les modifications en appuyant sur le bouton "Commit" toujours sur la "Side Bar"
4. Pusher les modifications en appuyant sur le bouton Push sur l'aire principale
5. Voilà!
